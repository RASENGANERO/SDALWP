== Description ==

Плагин для создания квизов и тестов в WordPress.

== Installation ==

1. Upload `quizle` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Changelog ==

* Добавлено: мессенджеры в сбор контактных данных

* Исправлено: вывод колонок с типом вопросов "Варианты ответов"

* Улучшили: мобильную версию в админ панели
